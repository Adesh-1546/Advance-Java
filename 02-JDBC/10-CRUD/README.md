# CRUD Operations in JDBC

This module demonstrates separate implementations of CRUD operations using JDBC.

---

## 🔑 Operations Covered

### 1. Create

Insert new record into database

---

### 2. Read

Fetch records from database

---

### 3. Update

Modify existing records

---

### 4. Delete

Remove records from database

---

## 🔥 Approach Used

* PostgreSQL database
* JDBC API
* PreparedStatement for all operations

---

## 🔥 Why Separate Files?

* Better readability
* Easier testing
* Clean project structure

---

## 🔥 Interview Questions

### 1. What is CRUD?

Basic operations: Create, Read, Update, Delete

---

### 2. Which class is used?

PreparedStatement

---

### 3. Why not Statement?

Security issues (SQL Injection)

---

## 🚀 Summary

CRUD operations are essential for any backend system and form the foundation for APIs and database interaction.
